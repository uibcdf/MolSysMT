molsysmt.basic.view#

molsysmt.basic.view(molecular_system=None, selection='all', structure_indices='all', standard=False, with_water_as=None, viewer='NGLView', syntax='MolSysMT')[source]#

Showing a molecular system.

This function assists the user in the usage of molecular visualization libraries (“viewer”) to show molecular systems in a Jupyter notebook.

Parameters:

  • molecular_system (molecular system) – Molecular system in any of the supported forms to be shown in the Jupyter notebook by the viewer.

  • selection (index, tuple, list, numpy.ndarray or str, default 'all') – Selection of atoms of the molecular system to be shown by the viewer. The selection can be given by a list, tuple or numpy array of atom indices (0-based integers); or by means of a query string following any of the selection syntaxes parsable by MolSysMT.

  • structure_indices (integer, tuple, list, numpy.ndarray or 'all', default 'all') – Indices of structures (0-based integers) to be shown by the viewer.

  • standard (bool, default False) –

    Option to standardize views with the following features:

    • sdf

  • with_water_as ({'licorice', 'surface', None}, default None) – Input argument to choose the water molecules representation.

  • viewer ({'NGLView'}, default 'NGLView') – Molecular visualization library to be used as viewer.

  • syntax (str, default 'MolSysMT') – Supported syntax used in the selection argument (in case it is a string).

Returns:

The function returns the visualization native object of the viewer.

Return type:

view native object of viewer

Raises:

  • NotSupportedFormError – The function raises a NotSupportedFormError in case a molecular system is introduced with a not supported form.

  • ArgumentError – The function raises an ArgumentError in case an input argument value does not meet the required conditions.

  • SyntaxError – The function raises a SyntaxError in case the syntax argument takes a not supported value.

New in version 0.1.0.

Notes

The list of supported molecular systems’ forms is detailed in the documentation section User Guide > Introduction > Molecular systems > Forms.

The list of supported selection syntaxes can be checked in the documentation section User Guide > Introduction > Selection syntaxes.

The list of supported viewers can be checked in the documentation section User Guide > Introduction > Viewers.

See also

molsysmt.basic.select()

Selecting elements of a molecular system

Examples

The following example illustrates the use of the function.

>>> import molsysmt as msm
>>> molecular_system = msm.systems.demo['T4 lysozyme L99A']['181l.mmtf']
>>> msm.basic.view(molecular_system, selection='molecule_type="protein"', viewer='NGLView')
nglview.NGLWidget()
