Color by value#

Coloring elements by scalar values

NGLView is a very flexible and versatile tool. Just have a look to its documentation, or to the wonderful Q&A discussion in the GitHub repository. Nervertheless, and in spite of being strictly unnecessary, MolSysMT wants to contribute with some shortcuts and tools. For instance, the color of elements such as atoms, or groups, can be set in a representation according to their scalar values in a list and a color map. The function molsysmt.thirds.nglview.color_by_value() is helpful for this purpose. Let’s see a couple of examples:

import molsysmt as msm

molecular_system = msm.convert('181L', selection='molecule_type=="protein"')

Once we have a protein, the charge of its aminoacids can be obtained with the function molsysmt.physchem.charge():

charge_residues = msm.physchem.charge(molecular_system, element='group', definition='physical_pH7')

Let’s make an NGLView view with a cartoon representation colored according to the amino acids charge. The first step is choosing, or making your own, matplotlib color map:

from matplotlib.pyplot import colormaps
blue_white_red = colormaps['bwr']

Tip

Have a look to the Matplotlib documentation: Choosing Colormaps in Matplotlib and Creating Colormaps in Matplotlib.

The second step is getting an NGLWidget object:

view = msm.view(molecular_system)
view.clear()

The colored representation can be then added to the view:

msm.thirds.nglview.color_by_value(view, charge_residues, element='group', cmap=blue_white_red, representation='cartoon')

And finally, the NGLView can be shown:

view

Let’s illustrate the use of molsysmt.thirds.nglview.color_by_value() with another example. This time, let’s color a licorice representation with the b-factor of each atom:

b_factors = msm.get(molecular_system, element='atom', b_factor=True)

white_blue_purple = colormaps['BuPu']
